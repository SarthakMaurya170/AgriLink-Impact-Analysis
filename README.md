# Agri-Link Impact Analysis Dashboard 

## Project Overview
This project analyzes the socio-economic impact of the "Agri-Link" initiative on agricultural and manufacturing commodities. By processing unstructured wage statistics, this analysis maps out the trajectory of worker population growth and evaluates how commission structures correlated with absolute wage increases.

## Objective
To clean and transform raw, unstructured Excel exports into relational datasets and extract actionable insights regarding wage growth before and after the implementation of the Agri-Link program.

## Key Insights
1. **Consistent Workforce Growth:** The worker population rate saw a steady multi-year climb, peaking at 62.1% in the 2023-24 financial year.
2. **Massive Wage Spikes:** Commodities like Coal mining, Sugar, and Milk Products saw absolute wage increases of over 200% following the Agri-Link integration.
3. **Commission Correlation:** Analyzed how baseline commission percentages (ranging from 20% to 32%) influenced the overall percentage of wage growth across different sectors. 

## Tech Stack
* **Language:** Python
* **Data Wrangling:** Pandas, NumPy (Handling multi-table unformatted CSVs, feature engineering)
* **Visualization:** Matplotlib, Seaborn
